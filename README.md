## 📋 Sobre o projeto

Sistema desenvolvido para facilitar o controle de estoque de uma distribuidora Rocha, permitindo o cadastro, edição, pesquisa e exclusão de **clientes** e **fornecedores** de forma simples e organizada.

---

## ✨ Funcionalidades

### 👤 Módulo Cliente
- ✅ Cadastrar cliente
- ✅ Editar cliente
- ✅ Excluir cliente
- ✅ Pesquisar cliente

### 🏭 Módulo Fornecedor
- ✅ Cadastrar fornecedor
- ✅ Editar fornecedor
- ✅ Excluir fornecedor
- ✅ Pesquisar fornecedor

### 🛡️ Validações
- Campos obrigatórios
- CPF e CNPJ (apenas números)
- Número de telefone
- Duplicatas (impede cadastro repetido)
- Alertas de sucesso, erro e aviso
